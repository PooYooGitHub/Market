## 2. MinIO 图片访问原理

### 2.1 图片上传流程

```
前端上传图片
    ↓
File服务(8106)
    ↓
MinIO存储(9000)
    ↓
返回图片URL: http://localhost:9000/market-product/abc123.jpg
    ↓
保存到数据库
```

### 2.2 图片访问流程

```
前端展示图片
    ↓
<img src="http://localhost:9000/market-product/abc123.jpg" />
    ↓
浏览器直接请求 MinIO(9000)
    ↓
MinIO 返回图片数据
    ↓
浏览器显示图片
```

### 2.3 重要说明

**图片 URL 格式**:
```
http://localhost:9000/{bucket-name}/{file-name}
```

**示例**:
```
http://localhost:9000/market-product/a1b2c3d4e5f6.jpg
http://localhost:9000/market-avatar/user123456.png
```

**前端可以直接使用这个 URL**:
```html
<!-- 直接在 img 标签中使用 -->
<img src="http://localhost:9000/market-product/a1b2c3d4e5f6.jpg" />

<!-- 在 CSS 中使用 -->
<div style="background-image: url('http://localhost:9000/market-product/a1b2c3d4e5f6.jpg')"></div>
```

**为什么可以直接访问?**
- MinIO 的桶已设置为**公开读**权限
- 任何人都可以通过 URL 直接访问图片
- 不需要 Token 认证

## 3. 接口列表

### 3.1 上传单张商品图片

#### 基本信息

- **接口地址**: `/api/file/upload/product`
- **请求方式**: `POST`
- **Content-Type**: `multipart/form-data`
- **是否需要登录**: ✅ 是（需要 Token）

#### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| file | File | ✅ | 图片文件 |

#### 请求头

```http
Content-Type: multipart/form-data
satoken: <用户登录后的token>
```

#### 响应示例

```json
{
  "code": 200,
  "message": "上传成功",
  "data": "http://localhost:9000/market-product/a1b2c3d4e5f6.jpg"
}
```

#### 前端使用示例

```javascript
// 使用 FormData 上传
const uploadImage = async (file) => {
  const formData = new FormData();
  formData.append('file', file);
  
  const token = localStorage.getItem('satoken');
  const response = await axios.post('/api/file/upload/product', formData, {
    headers: {
      'Content-Type': 'multipart/form-data',
      'satoken': token
    }
  });
  
  if (response.data.code === 200) {
    const imageUrl = response.data.data;
    console.log('图片URL:', imageUrl);
    // 可以直接用于 <img src={imageUrl} />
    return imageUrl;
  }
};
```

### 3.2 批量上传商品图片

#### 基本信息

- **接口地址**: `/api/file/upload/product/batch`
- **请求方式**: `POST`
- **Content-Type**: `multipart/form-data`
- **是否需要登录**: ✅ 是

#### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| files | File[] | ✅ | 图片文件数组（最多9张） |

#### 响应示例

```json
{
  "code": 200,
  "message": "上传成功",
  "data": [
    "http://localhost:9000/market-product/a1b2c3d4e5f6.jpg",
    "http://localhost:9000/market-product/b2c3d4e5f6g7.jpg",
    "http://localhost:9000/market-product/c3d4e5f6g7h8.jpg"
  ]
}
```

#### 前端使用示例

```javascript
// 批量上传
const uploadImages = async (files) => {
  const formData = new FormData();
  
  // 注意：所有文件都使用同一个 key "files"
  files.forEach(file => {
    formData.append('files', file);
  });
  
  const token = localStorage.getItem('satoken');
  const response = await axios.post('/api/file/upload/product/batch', formData, {
    headers: {
      'Content-Type': 'multipart/form-data',
      'satoken': token
    }
  });
  
  if (response.data.code === 200) {
    const imageUrls = response.data.data; // 数组
    return imageUrls;
  }
};
```

## 4. 前端完整示例

### 4.1 Vue3 + Element Plus 上传组件

```vue
<template>
  <div class="image-upload">
    <el-upload
      v-model:file-list="fileList"
      :action="uploadUrl"
      :headers="uploadHeaders"
      :on-success="handleSuccess"
      :on-error="handleError"
      :before-upload="beforeUpload"
      :limit="9"
      list-type="picture-card"
      accept="image/*"
    >
      <el-icon><Plus /></el-icon>
    </el-upload>
    
    <el-button type="primary" @click="handleSubmit">提交</el-button>
  </div>
</template>

<script setup>
import { ref, computed } from 'vue';
import { ElMessage } from 'element-plus';
import { Plus } from '@element-plus/icons-vue';

const fileList = ref([]);

// 上传地址
const uploadUrl = computed(() => {
  return 'http://localhost:9000/api/file/upload/product';
});

// 请求头（携带 Token）
const uploadHeaders = computed(() => {
  const token = localStorage.getItem('satoken');
  return {
    'satoken': token
  };
});

// 上传前校验
const beforeUpload = (file) => {
  // 1. 校验文件类型
  const isImage = file.type.startsWith('image/');
  if (!isImage) {
    ElMessage.error('只能上传图片!');
    return false;
  }
  
  // 2. 校验文件大小（5MB）
  const isLt5M = file.size / 1024 / 1024 < 5;
  if (!isLt5M) {
    ElMessage.error('图片大小不能超过 5MB!');
    return false;
  }
  
  return true;
};

// 上传成功回调
const handleSuccess = (response, file, fileList) => {
  if (response.code === 200) {
    ElMessage.success('上传成功');
    console.log('图片URL:', response.data);
    
    // 将 URL 保存到 file 对象中，方便后续使用
    file.url = response.data;
  } else {
    ElMessage.error(response.message || '上传失败');
  }
};

// 上传失败回调
const handleError = (error, file, fileList) => {
  console.error('上传失败:', error);
  ElMessage.error('上传失败，请重试');
};

// 提交表单
const handleSubmit = () => {
  // 获取所有已上传的图片 URL
  const imageUrls = fileList.value
    .filter(file => file.url) // 只要已上传成功的
    .map(file => file.url);
  
  if (imageUrls.length === 0) {
    ElMessage.warning('请先上传图片');
    return;
  }
  
  console.log('提交的图片URLs:', imageUrls);
  
  // 调用发布商品接口
  publishProduct(imageUrls);
};

// 发布商品（示例）
const publishProduct = async (imageUrls) => {
  try {
    const response = await axios.post('/api/product/publish', {
      title: '商品标题',
      description: '商品描述',
      price: 100,
      categoryId: 1,
      imageUrls: imageUrls  // 传递图片URL数组
    }, {
      headers: {
        'satoken': localStorage.getItem('satoken')
      }
    });
    
    if (response.data.code === 200) {
      ElMessage.success('发布成功');
    }
  } catch (error) {
    ElMessage.error('发布失败');
  }
};
</script>

<style scoped>
.image-upload {
  padding: 20px;
}
</style>
```

### 4.2 手动上传（不使用 el-upload）

```vue
<template>
  <div class="manual-upload">
    <input 
      type="file" 
      ref="fileInput"
      accept="image/*"
      multiple
      @change="handleFileChange"
    />
    <button @click="handleUpload">上传</button>
    
    <!-- 预览已上传的图片 -->
    <div class="preview">
      <img 
        v-for="(url, index) in uploadedUrls" 
        :key="index"
        :src="url" 
        alt="商品图片"
        style="width: 100px; height: 100px; margin: 5px;"
      />
    </div>
  </div>
</template>

<script setup>
import { ref } from 'vue';
import axios from 'axios';
import { ElMessage } from 'element-plus';

const fileInput = ref(null);
const selectedFiles = ref([]);
const uploadedUrls = ref([]);

// 文件选择
const handleFileChange = (event) => {
  const files = Array.from(event.target.files);
  
  // 校验数量
  if (files.length > 9) {
    ElMessage.warning('最多选择9张图片');
    return;
  }
  
  // 校验每个文件
  for (const file of files) {
    if (!file.type.startsWith('image/')) {
      ElMessage.error('只能上传图片');
      return;
    }
    if (file.size > 5 * 1024 * 1024) {
      ElMessage.error(`${file.name} 超过5MB`);
      return;
    }
  }
  
  selectedFiles.value = files;
};

// 上传文件
const handleUpload = async () => {
  if (selectedFiles.value.length === 0) {
    ElMessage.warning('请先选择文件');
    return;
  }
  
  try {
    // 创建 FormData
    const formData = new FormData();
    selectedFiles.value.forEach(file => {
      formData.append('files', file);
    });
    
    // 上传
    const token = localStorage.getItem('satoken');
    const response = await axios.post(
      'http://localhost:9000/api/file/upload/product/batch',
      formData,
      {
        headers: {
          'Content-Type': 'multipart/form-data',
          'satoken': token
        }
      }
    );
    
    if (response.data.code === 200) {
      uploadedUrls.value = response.data.data;
      ElMessage.success('上传成功');
      
      // 清空选择
      selectedFiles.value = [];
      fileInput.value.value = '';
    }
  } catch (error) {
    console.error('上传失败:', error);
    ElMessage.error('上传失败');
  }
};
</script>
```

### 5.4 为什么可以直接访问？

1. **MinIO 桶设置了公开读权限**
   - 桶策略配置为 `Public`
   - 允许匿名用户读取对象

2. **不需要签名认证**
   - 公开桶的图片可以直接通过 URL 访问
   - 不需要在请求中携带 Token

3. **浏览器可以直接请求**
   - 浏览器的 `<img>` 标签会直接向 MinIO 发起请求
   - MinIO 返回图片二进制数据
   - 浏览器渲染显示图片

### 5.5 访问流程示意图

```
1. 上传阶段:
前端 → File服务(8106) → MinIO(9000) → 返回URL

2. 访问阶段:
浏览器 <img src="http://localhost:9000/market-product/abc.jpg" />
   ↓
直接请求 MinIO(9000)
   ↓
MinIO 返回图片数据
   ↓
浏览器显示图片
```

**注意**: 图片访问**不经过**Gateway(9000端口是MinIO,不是Gateway)

## 6. 常见问题

### Q1: 图片上传后显示不出来？

**A**: 检查以下几点
1. 图片 URL 是否完整且正确
2. MinIO 是否正在运行 (`docker ps`)
3. 桶是否设置了公开读权限
4. 浏览器控制台是否有跨域错误

### Q2: 如何设置桶为公开读？

**A**: 在 File 服务启动时会自动设置，或手动设置：
```bash
# 登录 MinIO 控制台
http://localhost:9001

# 选择桶 → Access Policy → 设置为 Public
```

### Q3: 上传时提示401？

**A**: 需要登录并携带 Token
```javascript
headers: {
  'satoken': localStorage.getItem('satoken')
}
```

### Q4: 如何限制上传图片的尺寸？

**A**: 前端可以使用 FileReader 读取图片并检查
```javascript
const checkImageSize = (file) => {
  return new Promise((resolve) => {
    const reader = new FileReader();
    reader.onload = (e) => {
      const img = new Image();
      img.onload = () => {
        if (img.width > 2000 || img.height > 2000) {
          ElMessage.error('图片尺寸不能超过2000x2000');
          resolve(false);
        } else {
          resolve(true);
        }
      };
      img.src = e.target.result;
    };
    reader.readAsDataURL(file);
  });
};
```
