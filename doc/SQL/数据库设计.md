## 1. 用户服务数据库 (market_user)

### 1.1 用户表 (t_user)

| 字段名 | 类型 | 长度 | 必填 | 默认值 | 说明 |
|---|---|---|---|---|---|
| id | BIGINT | 20 | 是 | - | 主键ID，自增 |
| username | VARCHAR | 50 | 是 | - | 用户名，唯一 |
| password | VARCHAR | 100 | 是 | - | 密码（BCrypt加密） |
| nickname | VARCHAR | 50 | 否 | - | 用户昵称 |
| avatar | VARCHAR | 255 | 否 | - | 头像URL |
| phone | VARCHAR | 20 | 否 | - | 手机号，唯一 |
| email | VARCHAR | 100 | 否 | - | 邮箱 |
| status | TINYINT | 4 | 否 | 1 | 状态 1:正常 0:禁用 |
| create_time | DATETIME | - | 否 | CURRENT_TIMESTAMP | 创建时间 |
| update_time | DATETIME | - | 否 | CURRENT_TIMESTAMP | 更新时间 |

### 1.2 角色表 (t_role)

| 字段名 | 类型 | 长度 | 必填 | 默认值 | 说明 |
|---|---|---|---|---|---|
| id | BIGINT | 20 | 是 | - | 主键ID |
| role_name | VARCHAR | 50 | 是 | - | 角色名称 (如：管理员) |
| role_code | VARCHAR | 50 | 是 | - | 角色编码 (如：ROLE_ADMIN) |
| description | VARCHAR | 255 | 否 | - | 描述 |
| create_time | DATETIME | - | 否 | CURRENT_TIMESTAMP | 创建时间 |

### 1.3 用户角色关联表 (t_user_role)

| 字段名 | 类型 | 长度 | 必填 | 默认值 | 说明 |
|---|---|---|---|---|---|
| id | BIGINT | 20 | 是 | - | 主键ID |
| user_id | BIGINT | 20 | 是 | - | 用户ID |
| role_id | BIGINT | 20 | 是 | - | 角色ID |
